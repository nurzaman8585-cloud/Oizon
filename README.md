# OIZON — маркетплейс электроники

## Структура сайта

| Файл | Страница |
|---|---|
| `index.html` | Главная (витрина) |
| `catalog.html` | Каталог — Ноутбуки |
| `catalog-printer.html` | Каталог — Принтеры |
| `catalog-cartridge.html` | Каталог — Расходники для принтеров |
| `catalog-gadget.html` | Каталог — Гаджеты |
| `catalog-accessory.html` | Каталог — Аксессуары |
| `catalog-sale.html` | Акции — только товары со скидкой из всех категорий |
| `product.html` | Карточка товара — Ноутбук ProBook 15" |
| `product-printer.html` | Карточка товара — МФУ LaserJet 420 |
| `product-cartridge.html` | Карточка товара — Картридж CF283A |
| `product-gadget.html` | Карточка товара — Смарт-часы FitTrack 2 |
| `product-accessory.html` | Карточка товара — Мышь SilentClick |
| `cart.html` | Корзина → заказ уходит в WhatsApp |
| `brand/` | Логотип и иконки в SVG |

Как открыть: просто открой `index.html` в браузере — все переходы между страницами уже работают.

## Как устроена навигация

- Каждая категория (Ноутбуки/Принтеры/Расходники/Гаджеты/Аксессуары) имеет свою
  страницу-каталог со списком товаров и фильтрами.
- В каждом каталоге пока только **один товар кликабелен по-настоящему** (ведёт на
  готовую карточку товара) — остальные карточки в сетке пока витринные заглушки
  с теми же характеристиками оформления. Когда появятся реальные товары под каждую
  категорию, для них нужно будет по образцу существующих product-*.html сделать
  отдельные карточки и проставить им ссылки в каталогах.
- Страница «Акции» не отдельная категория, а фильтр по всем категориям — туда
  попадают товары, у которых в карточке есть скидка (`price-old` рядом с ценой).
  При добавлении новых акционных товаров нужно вручную добавить их сюда же.

## Что нужно сделать перед реальным запуском

1. **Номер WhatsApp** — в `cart.html` найди строку
   `const whatsappNumber = '996700000000';`
   и замени на настоящий номер (без «+» и пробелов).

2. **Реальные фото товаров** — сейчас везде вместо фото серые заглушки-иконки.
   В карточках товаров в блоках `<div class="desc-image">` и `<div class="gallery-stage">`
   достаточно вставить обычный `<img src="...">` — стили уже настроены.

3. **Карточки под остальные товары в каталогах** — см. пункт выше про навигацию.

4. **Интеграция с МоимСкладом** — сейчас все цены, остатки и товары зашиты в HTML
   вручную. Договорились делать это в самом конце, когда структура сайта будет
   полностью готова — тогда карточки и каталоги станут подтягивать данные по API
   автоматически, а «Акции» смогут собираться сами по флагу скидки в системе.

5. **Хостинг** — сейчас это просто набор HTML-файлов. Чтобы сайт был доступен
   по ссылке в интернете, файлы нужно загрузить на хостинг (например, обычный
   веб-хостинг, GitHub Pages, Netlify, Vercel — подойдёт любой, который отдаёт
   статичные файлы).
